# Levi's STR Floorplan Editor

A React web app for annotating apartment floorplan images with emergency evacuation routes and safety information.

## Features
- Upload floorplan images
- Draw primary (red) and secondary (yellow) evacuation routes with directional arrows
- Place safety icons: fire extinguisher, first aid kit, smoke alarm
- Add EXIT markers and occupancy labels
- Editable legend
- Export annotated floorplan as PDF (matching original image dimensions)
- Import LLM-generated JSON annotations

## Tech Stack
- React + Vite
- HTML5 Canvas
- jsPDF (via CDN)

## Getting Started
```
npm install
npm run dev
```
